## 4. Key Findings

### ✅ Working Well
- **BJJ submissions active** — 86.6% of BJJ wins are by submission. Bottom bonus fix worked
- **Wrestler viable** — 66%+ win rate as player, good balance between KO/Sub/Dec
- **All-Rounder balanced** — 64.7% overall, closest to ideal. 10.8% KO / 11.2% Sub / 42.7% Dec
- **Simulation runs fast** — 80,000 fights in 46 seconds

### 🔴 Needs Attention
- **Muay Thai 50.5% KO rate** — too high. Every other fight ends by KO
- **Boxer 100% vs Wrestler** — should be reversed (Wrestler should counter Boxer)
- **Muay Thai 99%+ vs non-BJJ fighters** — complete domination
- **Player advantage asymmetry** — archetype-aware exchange ordering gives player huge advantage. Mirror matchups show 60-95% disparity

### 🟡 Structural Notes
- **Archetype-aware exchange ordering** — first 2 exchanges favor player's archetype. This creates large asymmetry between "A as player vs B" and "B as player vs A"
- **Striking is still king** — fights start standing, strikers land damage before grapplers can takedown
- **Base stats ~60** — simulation uses balanced base stats. Real game fighters will have varied stats from training, making matchups more unpredictable
