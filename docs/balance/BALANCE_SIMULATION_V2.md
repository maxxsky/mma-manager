## 6. Key Findings (v2)

### ✅ Fixed
- **A/B bias removed** — exchange-opening now uses `pickExchange()` uniformly for all exchanges
- **matchupMod bidirectional** — both A and B get their archetype bonuses from the matchup table
- **Mirror matchups now asymmetric by design** — different bonuses for A vs B depending on matchup direction (rock-paper-scissors)

### 🔴 Remaining Balance Issues (stats-based, not bias)
- **Muay Thai 48.1% KO rate** — striking ×1.08 + clinch bonus still dominant
- **Boxer 100% vs Wrestler** — striking ×1.15 + footwork ×1.10 overwhelms Wrestler before takedown
- **Striking archetypes dominant at equal base stats** — fights start standing, grapplers need higher base stats to compete

### 🟡 Recommended Next Steps (if balancing desired)
- Reduce Boxer striking from 1.15 → 1.10, footwork from 1.10 → 1.05
- Reduce Muay Thai striking from 1.08 → 1.05, clinch bonus from 1.15 → 1.10
- Increase Wrestler durability: add cardio or chin bonus
- Or: increase all grappler base wrestling to make takedowns more reliable
